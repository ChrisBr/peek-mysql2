## Installation

Add this line to your application's Gemfile:

    gem 'glimpse-mysql2'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install glimpse-mysql2

## Usage

Add the following to your `config/initializers/glimpse.rb`: 

```ruby
Glimpse.into Glimpse::Views::Mysql2
```
